M MoveTo. Used to start a new path. Can also be used during a path to move without drawing.
H, V Horizontal and Vertical.
L LineTo. Draws a line in any direction.
Q Quadradic Bezier curve. Press once to target the contour, a second time to create the curve.
Z Draws from the current position back the the last created M. Useful for closing a path.
Esc Drops selection on the current path
1-9 Selects a path
